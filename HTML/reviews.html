Review on Some Useful Web Related Resources

Comments on W3Schools

W3Schools is an educational website for learning web technologies online.Content includes tutorials and references relating to HTML, CSS, JavaScript, JSON, PHP, Python, AngularJS, SQL, Bootstrap, Node.js, jQuery, XQuery, AJAX, XML, and Java.W3schools is very useful site for beginers and professional both in the field of web development. Personally, I have visited W3schools many times to solve my problems as well as to learn something new and W3Schools always provides us with the best contents.W3Schools includes the feature of TryIt editor which enables users to write their code on the browser and see the results right there, which is very impressive and useful for beginners like us.

Comments on CodePen

CodePen is a web based platform where developers can write their code and see the instant results in the browser and also can get feedbacks from other visitors and deelopers.CodePen mainly focuses on Front-End languages like HTML, CSS and JavaScript.It is a very good platform to develop web development skills since it is a open source community. Developers can interact with each other and solve the problems and make the design more efficient and more professional.CodePen can also be used for prototyping new ideas, reduced test cases for bugs, sending clients things to look at, evaluating potential hires, to finding inspiration. It can also be used as a sort of resume and portfolio, showing off their best design and development work.And in my opinion it is the best for doing these stuffs.

Comments on Font Awesome

Font Awesome is an icon library with thousands of icons. It contains all social media icons, action icons, notation icons, logos and many more.It is very useful to the web developers wishing to make webpages more attractive. With the use of icons the site looks more clear and attractive. My experience with font awesome icons is very much interesting. I have used icons from font-awesome library whenever I am designing a website. Using Font-Awesome icons library is also very easy. You can simply add the library css to your html file and you can use all icons in the libarry with defined syntax.
